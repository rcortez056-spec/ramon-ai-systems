# 🤖 Agentic AI Workflow Dashboard

A premium, interactive dashboard built specifically for AI Architects and Product Managers to visualize multi-agent orchestration and live reasoning feeds.

## 🎨 Included Color Themes

This asset kit comes pre-configured with three high-end color profiles tailored for modern enterprise and technical presentations:

1. **Cyberpunk Neo-Dark (Active Default)** – High-contrast terminal neon accents designed for high-impact visual monitoring.
2. **Enterprise Sleek Cardinal** – Professional slate and deep corporate blue tones built for executive dashboard reviews.
3. **Matte Obsidian & Gold** – Ultra-premium, low-contrast luxury aesthetic emphasizing refined minimalist dark styling.

## 🛠️ How to Swap Themes

Switching the visual profile of your dashboard takes less than 30 seconds:

1. Open `index.html` in your favorite code editor.
2. Locate the active `:root` CSS block near the top of the file (lines 11–24).
3. Scroll down to your desired theme block inside the CSS comments (Lines 25–48).
4. Copy the variable values of your chosen theme and paste them over the active variables inside the `:root` block.
5. Save the file, and your new dashboard skin is live!
